<p dir="rtl">
قواعد الكتابة</p>



<p dir="rtl">
اضافه الصور</p>
 
<p dir="rtl">
طريقة اضافة الصور</p>



```bash
Image("Background")
إسم الصورة 
```

<p dir="rtl">

تعديل الصورة</p>

<p dir="rtl">
طريقة تعديلها</p>



```bash
.resizable()
.scaledToFit()
.frame(width:#, height:#, alignment: .center)
.clipShape(Circle())
```




---

<p dir="rtl">
التمرين</p>




1. اضف ٣ صور أفلامك المفضلة 
2. اعرض الصوره بستخدام scaledToFit()
3. اعرض فيلمين آخرين أسفلها بشكل دائري
